Приложение 1: Словарь терминов

Возбуждение — состояние сексуального возбуждения, которое возникает в ответ на стимуляцию.

Эякуляция — процесс выброса спермы из полового члена.

Преждевременная эякуляция (ПЭ) — эякуляция, которая происходит раньше, чем хотел мужчина.

Точка невозврата (ПОНР) — момент, после которого эякуляция становится неизбежной.

Модель "Банка" — модель, которая описывает возбуждение как накопление в "банке", которая может переполниться.

Модель фаз — модель, которая описывает эякуляцию как процесс, состоящий из двух фаз: эмиссии и изгнания.

Модель триггеров — модель, которая описывает факторы, которые вызывают эякуляцию.

Модель столпов — модель, которая описывает четыре столпа мастерства: осознанность, дыхание, расслабление, управление.

Осознанность — способность замечать процесс, изменения в теле, рост возбуждения.

Дыхание — инструмент управления возбуждением через активацию парасимпатической нервной системы.

Расслабление — создание пространства для контроля через снижение напряжения.

Управление — активное влияние на процесс через использование техник.

Старт-стоп — техника, которая включает остановку и возобновление стимуляции.

Сжатие — техника, которая включает физическое сжатие головки пениса для остановки процесса.

ЛК-мышцы (PC-мышцы) — лобково-копчиковые мышцы, которые контролируют эякуляцию.

ОМ (Осознанная мастурбация) — практика мастурбации с фокусом на осознанности и контроле.

Интеграция — объединение всех изученных элементов в единую систему.
