# Аудио Джаана

Сюда скидывай записанные MP3/M4A файлы перед загрузкой в Supabase.

После того как скинул файл — говори мне (в Cursor):
> "Загрузи audio/welcome.mp3 как onboarding_welcome"

Я запущу скрипт, загружу в Supabase и обновлю приложение автоматически.

## Ключи для треков

| Ключ | Что это |
|------|---------|
| `onboarding_welcome` | "Привет. Я — Джаан." — первый экран |
| `onboarding_path` | "Что тебя ждёт" — про путь |
| `onboarding_before_diagnostics` | Перед диагностикой |
| `home_daily_greeting` | Ежедневное приветствие на главной |
| `day_1` ... `day_30` | Вступление к каждому дню |
| `checkpoint_week1` | После первой недели |
| `checkpoint_week2` | После второй недели |
| `checkpoint_week3` | После третьей недели |
| `checkpoint_final` | Финал курса |
| `ai_trainer_welcome` | Первое открытие чата с тренером |
| `intro_1` | Вводный модуль 1 — "Откуда это взялось" |
| `intro_2` | Вводный модуль 2 — "Как устроен мой метод" |
| `intro_3` | Вводный модуль 3 — "Что тебя ждёт в 30 днях" |
| `stories.story_01` | Сказка 1 (название запишешь сам) |
| `stories.story_02` | Сказка 2 |
| `stories.story_03` ... `story_10` | Сказки 3–10 |
